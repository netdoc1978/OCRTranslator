# OCR识别翻译工具 v1.0

## 功能特性

- **图片识别**：支持 JPG、PNG、BMP、GIF 等常见图片格式的 OCR 文字识别
- **PDF文档识别**：支持 PDF 文档的页面识别
- **截图识别**：按 F4 快捷键快速截图识别（支持自定义区域选择）
- **多语言翻译**：支持中文、英文、日文、韩文、法语、西班牙语等 20+ 种语言的翻译
- **开机自启动**：可选开启开机自动运行
- **系统托盘**：关闭主窗口后程序最小化到系统托盘，不占用任务栏

## 系统要求

- Windows 7 及以上版本
- .NET Framework 4.0 或更高版本（Windows 10/11 自带）
- 网络连接（用于调用百度 OCR 和翻译 API）

## 使用方法

### 基本操作

1. **打开图片**：点击"打开图片"按钮，选择本地图片文件
2. **打开PDF**：点击"打开PDF"按钮，选择 PDF 文档
3. **粘贴图片**：从剪贴板粘贴图片（Ctrl+V）
4. **截图识别**：按 F4 快捷键，然后拖动鼠标选择识别区域

### 识别设置

- **识别类型**：选择文字语言类型（中英文混合、中文、英文、日文、韩文）
- **自动翻译**：勾选后识别完成自动翻译

### 翻译设置

- **翻译语言**：选择目标翻译语言
- 支持语言：中文、繁体、英语、日语、韩语、法语、西班牙语、俄语、德语等

### 保存结果

- **复制原文/译文**：点击按钮复制到剪贴板
- **保存**：将识别和翻译结果保存为文本文件

## 快捷键

- **F4**：截图识别（全局快捷键，程序在后台时也可使用）
- **ESC**：取消截图操作

## 系统托盘功能

- 关闭主窗口：程序最小化到系统托盘
- 双击托盘图标：显示主窗口
- 右键菜单：显示主窗口、截图识别、退出程序

## 注意事项

- 首次使用需要网络连接以获取 API Token
- F4 快捷键可能被部分安全软件拦截
- PDF 识别功能依赖系统 PDF 预览组件，如不可用请使用截图识别
