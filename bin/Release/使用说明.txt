OCR识别翻译工具 v1.0
==================================================

一、简介
这是一款小巧、简单、实用的Windows端OCR识别和翻译软件，基于百度智能云API开发。

二、主要功能
1. 图片识别 (OCR)
   - 支持打开本地图片进行识别
   - 支持粘贴剪贴板图片
   - 支持F4快捷键截图识别
   - 识别语言：中英文混合、中文、英文、日文、韩文
   - 识别模型：通用识别、高精度识别

2. 翻译功能
   - 支持14种翻译方向
   - 翻译类型：通用文本翻译，大模型文本翻译
   - 支持语言：中文、英文、日文、韩文、法文、西班牙文、俄文、德文等

3. 其他功能
   - 开机自启动
   - 窗口置顶
   - 识别后自动复制到剪贴板
   - 系统托盘运行

三、系统要求
- 操作系统：Windows 7 及以上版本
- .NET Framework 4.0 或更高版本
- 网络连接（用于调用百度API）

四、使用方法

1. 打开图片识别
   1) 点击"打开图片"按钮
   2) 选择本地图片文件（支持 JPG、PNG、BMP、GIF 格式）
   3) 程序将自动进行OCR识别
   4) 识别结果将显示在"识别文本"框中，并自动复制到剪贴板

2. 粘贴图片识别
   1) 在其他程序中复制图片到剪贴板
   2) 点击"粘贴图片"按钮
   3) 程序将自动进行OCR识别

3. 截图识别（推荐）
   1) 按 F4 键或点击"截图识别"按钮
   2) 拖动鼠标选择要识别的区域
   3) 松开鼠标完成截图
   4) 程序将自动进行OCR识别

4. 翻译
   1) 在"识别文本"框中确认或编辑要翻译的文本
   2) 选择翻译方向（如：英中、中英等）
   3) 选择翻译类型（通用文本/大模型文本）
   4) 点击"翻译"按钮
   5) 翻译结果将显示在"翻译结果"框中

5. 复制结果
   - 点击"复制原文"按钮复制识别文本
   - 点击"复制译文"按钮复制翻译结果
   - 识别完成后文本会自动复制到剪贴板

五、快捷键
F4  截图识别

六、界面说明
程序窗口分为左右两部分：
- 左侧：图片预览、三个功能按钮（打开图片、粘贴图片、截图识别）、设置选项
- 右侧：上方为识别文本区域，下方为翻译结果区域

七、托盘图标
程序启动后会自动最小化到系统托盘：
- 双击托盘图标：显示主窗口
- 右键托盘图标：显示快捷菜单
  - 显示主窗口
  - 截图识别
  - 退出

八、注意事项
1. 程序需要联网才能使用OCR识别和翻译功能
2. 百度API有免费额度限制，请合理使用
3. 截图识别是最可靠的识别方式
4. 识别效果取决于图片清晰度和文字对比度

九、常见问题

问：为什么识别失败？
答：请检查：
  - 网络连接是否正常
  - 图片是否清晰
  - 百度API额度是否用完

问：如何使用PDF文件？
答：请使用截图识别（F4）功能，在PDF阅读器中截取需要识别的区域

问：如何设置开机自启动？
答：勾选界面左下角的"开机自启动"选项即可

问：为什么有些内容没有被翻译？
答：百度翻译API可能会将某些专有名词、文件名等识别为无需翻译的内容。可以尝试使用"大模型文本翻译"选项。
